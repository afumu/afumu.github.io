# AI源计划 - 《Midjourney绘画教程》

首先登录网站后，点击左侧菜单栏的 - > 专业绘画，绘画主要分为以下4步：

# 第一步

选择对应的图片尺寸，1:1为头像尺寸，4:3为公众号配图尺寸，3:4为社交媒体配图尺寸，16:9为电脑壁纸尺寸，9:16为海报尺寸。

# 第二步

选择模型，Midjourney 为写实风格，Niji为漫画风格。

# 第三步

选择参数，

Midjourney 可选的是：

混乱：混乱级别取值范围：0-100、 --chaos 或 --c 混乱级别，可以理解为让AI天马行空的空间，值越小越可靠、默认0最为精准。

风格化：风格化：--stylize 或 --s，范围 1-1000 参数释义：数值越高，画面表现也会更具丰富性和艺术性。

Niji可选的是：

混乱：混乱级别取值范围：0-100、 --chaos 或 --c 混乱级别，可以理解为让AI天马行空的空间，值越小越可靠、默认0最为精准。

# 第四步

输入你要生成的提示词，点击提交即可，平台自带翻译功能，无需自己再去翻译一次。例：

马戏团模型提示词： playground, circus, lion, tiger, monkey (opens new window), Low poly model in game asset style, isometric, tilt-shift, 16 bit pixel art, mockup (opens new window), fine glossy, white background, 3d render, oc render, best quality, 4k, ultra detailed

提醒：如果想生成头像，可以选择Midjourney ，如果想生成漫画或二次元风格，可以选择Niji，推荐选择最新的版本生成图片。

*在生成图片过程中，遇到问题可随时在群里进行提问，工作人员会帮解决问题！
