# 文生图

点击文生图，系统可以根据提示词生成对应的图片

# 模型选择

选择 Midjourney 生成的是真实风格的图片

现在 Niji 生成的是动漫风格的图片

# 版本

默认使用V6的版本即可

# 图片比例

这里可以选择生成的图片比例

# 品质

这里可以选择生成图片的质量，默认情况下为高清

# 混乱程度

取值范围：0-100、 --chaos 或 --c

混乱级别，可以理解为让AI天马行空的空间

值越小越可靠、默认0最为精准

# 风格化程度

风格化：--stylize 或 --s，范围 1-1000参数释义：数值越高，画面表现也会更具丰富性和艺术性

# seed

种子数是为每个图像随机生成的，如果您使用相同的种子号和提示，您将获得相似的最终图像。范围为0–4294967295的整数。

您可以在右侧找到喜欢的图片获取到此值！

# 禁用指令

开启后选项中的配置参数将不会携带，您可以在描述中进行自定义参数指定。

反之不打开、自定义输入--指令将会被忽略覆盖。

开启后不会翻译与优化咒语！
